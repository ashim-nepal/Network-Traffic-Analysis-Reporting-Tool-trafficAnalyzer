# Network-Traffic-Analysis-Reporting-Tool-trafficAnalyzer
# 

This simple Network Traffic Analysis Report Tool focuses on network visibility and analysis performs the following:

- Understand how network traffic flows
- Read and analyze captured packets from Wireshark
- Analyze protocol-level traffic using tshark
- Automate analysis using Python
- Generate a readable traffic analysis report


And finally, report is saved automatically to a report file.

## FAQ

#### What is this project about?

This project is a simple file that can be used from terminal using python that first read packet capture file ".pcap" and analyzes counts of total packets, identifies common protocols(DNS, HTTP, ICMP), highlights basic observations (unencrypted traffic) and generates a timestamped analysis report.



#### What programming language is used for Algorithm?

The specific language used for this project is Python and its Libraries.
